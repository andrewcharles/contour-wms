Installation
--------------------------------------------------------------------------------------------------------------------------

1. Install Python 32 Bit

2. Setup easy install for windows. The easiest way to do it is to run thet downloaded the ez_setup.py file.

3. Add C:\Python27\Scripts to the system path ( ENV vairables )

4. Easy install flask (cmd : easy_install Flask)

5. Install Numpy

6. Install Matplot lib

7. Install Basemap

8. Easy install pydap (cmd : easy_install pydap)
